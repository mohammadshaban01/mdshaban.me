Job Description

QUALITY ANALYST

JOB SUMMARY

The analyst will be responsible for testing and confirming the quality of the delivery of the web and backend transactional operations. This position is on the QA team creating a suite of automated test for regression on a Meanstack application layer. This role will require a high degree of technical aptitude, with a strong focus on programming skills and understanding the business logics.

SKILL SET REQUIRED

SDLC life cycle , UNIX, NoSQL, MongoDB.Hands on Tools – QTP, Selenium, DAC tools, Querysurge, JMeter.
Should have some experience with Hadoop, HDFS, SQL queries, Excel, ETL testing.Automation testing in Big data, Frontend(Angular), cross device and browser testing.
Good front end testing experience for Responsive applications across various platforms.

QUALIFICATION

B.Tech in Computer Science / Electrical Engineering.
